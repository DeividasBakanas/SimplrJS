Simplr Layout Examples

You call Simplr.Layout.mAssembleLayout() to create layouts.

Typically Layouts are used to create the html returned in views. In these examples I'm not using views.

The Layouts are pulled from the HTML document the first time they are used and then can be used as many times as needed afterward.
